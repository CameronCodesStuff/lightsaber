🚀 Getting Started
Clone the repository
```bash
git clone https://github.com/your-username/lightsaber.git
cd lightsaber
```
Launch locally
Simply open:
```bash
index.html
```
or use:
```bash
python -m http.server
```
Then visit:
```text
http://localhost:8000
```
